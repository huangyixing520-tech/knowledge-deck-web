# AI 协作入口

本项目采用团队 AI-DLC 文档驱动协作范式。AI 参与需求、设计、技术方案、开发、测试、评审、上线或复盘任务前，必须先阅读并遵循：

- [docs/AI_DLC.md](docs/AI_DLC.md)
- [docs/TEAM_SDLC.md](docs/TEAM_SDLC.md)

如果外部团队规范链接无法访问、看到登录页、权限页、空白页，或无法确认规范正文，必须明确说明“无法读取团队规范正文，请提供内容”，不得自行猜测。

除非任务非常简单，否则输出需要包含当前阶段、当前理解、缺失信息 / 假设、范围锁定、执行计划、风险与冲突、验收标准、验证方式、决策留痕和文档沉淀建议。
